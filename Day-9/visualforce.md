# Legacy Salesforce UI Framework and Historical Understanding

## Legacy Salesforce UI Framework

* Before LWC, Salesforce mainly used Visualforce and Aura Components for UI development.
* Visualforce was used to create custom pages using Apex and HTML-like tags.
* Aura Components introduced component-based development in Salesforce.
* These frameworks helped build dynamic Salesforce applications before LWC.

### Visualforce

* Page-centric framework.
* Uses Apex controllers for backend logic.
* Suitable for older Salesforce applications.

### Aura Components

* Component-based framework.
* Uses Aura markup and JavaScript controllers.
* Improved reusability compared to Visualforce.

---

# Historical Understanding

* Salesforce UI development evolved over time for better performance and user experience.
* Visualforce was powerful but slower and less modern.
* Aura improved flexibility but had complex architecture.
* Salesforce introduced LWC using web standards for faster and simpler development.
* Understanding older frameworks helps developers maintain legacy applications.

---

# Importance

* Many organizations still use Visualforce and Aura applications.
* Developers may need to support or migrate older systems.
* Historical knowledge helps understand Salesforce evolution and architecture.
* Useful for upgrading applications from Aura or Visualforce to LWC.
