# Salesforce Basics – Short Notes

## What is an App?
In Salesforce, an App is a collection of tools, tabs, and features designed for a specific business function.

### Examples
* Sales App
* Service App
* Marketing App
Apps help users access related features in one place.
---

## What is an Object?
An Object is a database table in Salesforce used to store data.

### Types
* Standard Objects → Account, Contact, Opportunity
* Custom Objects → User-created objects

### Object Contains
* Records → Rows of data
* Fields → Columns of data
---

## How UI is Structured
Salesforce UI (User Interface) is organized into:

### 1. App Launcher
Used to open apps and tools.

### 2. Navigation Bar
Displays tabs like Accounts, Contacts, Reports.

### 3. Record Page
Shows details of a selected record.

### 4. Tabs
Provide access to different objects and features.

### 5. Home Page
Main dashboard showing tasks, reports, and updates.
