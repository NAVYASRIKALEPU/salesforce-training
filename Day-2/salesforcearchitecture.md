## Multi-Tenant Architecture
Salesforce uses a multi-tenant architecture where multiple customers share the same infrastructure and servers while keeping their data secure and separate.

### Benefits
* Lower cost
* Automatic updates
* High scalability
* Secure data separation

# How Salesforce Platform Works
Salesforce works on a cloud platform where:

1. Data is stored online
2. Users access it through a web browser
3. Apps, automation, and reports run on the same platform
4. Businesses can customize features based on needs

### Main Components
* Objects → Store data
* Apps → Organize features
* Flows → Automate tasks
* Reports & Dashboards → Analyze data
