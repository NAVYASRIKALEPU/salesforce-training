# Lightning Web Components (LWC)

## What is LWC

* Lightning Web Components (LWC) is a modern framework used to build user interfaces in Salesforce.
* It is based on standard web technologies like HTML, JavaScript, and CSS.
* LWC provides fast performance and reusable components.
* Used to create responsive and interactive Salesforce applications.
* It is the preferred framework for modern Salesforce development.

---

## Component Structure

An LWC component mainly contains three files:

1. **HTML File**

   * Defines the user interface structure.

2. **JavaScript File**

   * Handles logic, data, and functionality.

3. **CSS File**

   * Adds styling to the component.

Example Structure:

```plaintext
helloWorld/
│── helloWorld.html
│── helloWorld.js
│── helloWorld.css
│── helloWorld.js-meta.xml
```

* The `.js-meta.xml` file defines component configuration and visibility in Salesforce.

---

## Modern Salesforce UI

* LWC helps create modern, fast, and responsive user interfaces.
* Uses reusable components for efficient development.
* Provides better performance compared to older frameworks.
* Supports dynamic and interactive web applications.
* Integrates easily with Salesforce data and services.
* Enhances user experience with clean and professional UI design.
