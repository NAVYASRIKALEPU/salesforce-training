# Salesforce Flow Builder – Short Notes
## What is Flow Builder?
Salesforce Flow Builder is a Salesforce tool used to automate business processes using a drag-and-drop interface with little or no coding.

### Uses
* Automate repetitive tasks
* Update records automatically
* Send emails and notifications
* Create guided workflows
---

# Playground Setup
A **Salesforce Playground** is a free practice environment provided by Salesforce to learn and test features safely.

### Setup Steps
1. Login to Trailhead
2. Open a module
3. Click **Launch**
4. Playground opens automatically

---
# Basic Automation Understanding
Automation means performing tasks automatically without manual work.

### Examples
* Auto-send emails
* Create tasks automatically
* Update records based on conditions

### Benefits
* Saves time
* Reduces errors
* Improves productivity
