# Salesforce Flow Overview, Automation Concepts, and Beginner-Friendly Flow Understanding

## Salesforce Flow Overview

* Salesforce Flow is a low-code automation tool used to automate business processes.
* It helps users perform actions automatically without writing complex code.
* Flow Builder provides a drag-and-drop interface for creating flows.
* Flows can collect user input, update records, send emails, and automate tasks.

### Common Types of Flows

* Screen Flow
* Record-Triggered Flow
* Scheduled Flow
* Autolaunched Flow

### Benefits

* Saves time
* Reduces manual work
* Improves productivity
* Automates business operations

# Beginner-Friendly Flow Understanding

* A flow works like a step-by-step process.
* Each step performs a specific action.
* Flow starts with a trigger or user action.
* Logic and conditions decide the next step.
* The flow ends after completing all actions.

### Simple Flow Example

1. User creates a record.
2. Flow checks conditions.
3. Email notification is sent.
4. Record gets updated automatically.

### Basic Structure

Start → Decision → Action → End
