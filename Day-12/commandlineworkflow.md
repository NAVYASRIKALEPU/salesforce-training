# Command-Line Workflow and Developer Productivity

## Command-Line Workflow

* Command-line workflow uses Salesforce CLI to manage development tasks.
* Developers can create projects, manage orgs, run tests, and deploy code using commands.
* Helps automate repetitive tasks and improves efficiency.
* Commonly used with Salesforce DX and VS Code.

### Common CLI Tasks

* Create scratch orgs
* Retrieve and deploy metadata
* Run Apex tests
* Authorize orgs
* Push source code

### Example Commands

```bash id="7dxw5u"
sfdx force:org:list
```

```bash id="x5c2hn"
sfdx force:source:push
```
