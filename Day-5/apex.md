# Apex – Short Notes

## Why Apex Exists
Apex is Salesforce’s programming language used to handle complex business logic that cannot be done using clicks alone.

### Uses
* Advanced automation
* Custom validations
* Integrations
* Complex calculations
---

# How Apex Works
* Apex runs on the Salesforce cloud platform
* It executes when triggered by events like record creation or button clicks
* Works with Salesforce data and objects

### Common Features
* Object-oriented language
* Similar to Java
* Runs securely in Salesforce environment
---

# Hello World Example
```apex
System.debug('Hello World');
```

### Explanation
* `System.debug()` prints output in Salesforce debug logs.
* Used for testing and debugging Apex code.
